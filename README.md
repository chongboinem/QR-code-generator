# QR-code-generator
QR code generation using Python
- Allows users to generate a quick QR code 
- Takes user input
- Allows users to customize how the QR code looks

**Set Up:**
  - choose a python code editor (vscode)
  - create a new file for the code
  - install 2 libraries : qrcode library and pillow library.
    to do that, run this command in vs code terminal:
        pip install qrcode pillow


**Run the code on git bash using: python file_name.py**
